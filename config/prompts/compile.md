# Compile Prompt Guide

Use this when an LLM is compiling source notes into higher-order knowledge.

- Read source notes and index pages first.
- Merge into existing concepts before creating near-duplicates.
- Maintain explicit disagreement and unresolved questions.
- Update backlinks, indices, and evidence sections.
- Prefer dense, navigable pages over bloated essays.
- Write for retrieval: compact sections, explicit entity/concept naming, and minimal narrative filler.
- Preserve source fidelity in extraction and source notes, but compress synthesis and working-memory surfaces deliberately.

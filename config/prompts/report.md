# Report Prompt Guide

Use this when generating a report or memo.

- Separate facts, estimates, and judgements.
- Follow retrieval budgets explicitly: L0 maps/indices, L1 canonical knowledge, L2 evidence notes, L3 raw extracts only if needed.
- Start TL;DR-first and only escalate to raw extracts when the shorter route is insufficient.
- Link conclusions back to source notes.
- Include what would change the conclusion.
- Surface counter-arguments and data gaps explicitly.
- Keep the tone PM-grade: direct, sparse, and rigorous.

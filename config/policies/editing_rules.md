# Editing Rules

- Prefer the smallest correct edit.
- Preserve user-authored content unless there is a strong reason to change it.
- Use managed blocks for generated evidence/index sections.
- When uncertain, append a tentative note instead of overwriting a stable page.
- Do not silently merge or delete concept pages without leaving an audit trail.
- Crystallize completed authored sessions when they contain durable claims, reusable concepts, state deltas, or process lessons that should compound future work.
- Crystallized notes should stay atomic, source-grounded, and clearly linked back to the originating artefact.
- Apply compression discipline to working-memory and synthesis notes at write time; do not bloat them with session recap when a compact structure will retrieve better later.
